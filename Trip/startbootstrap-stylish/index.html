Our RESTful API is build to today's web.

The Blackbaud NXT REST APIs allow you to manage constituent data such as contact info, education, relationships, attributes and constituent codes. Since the API is based on REST principles, it's very easy to write and test applications. You can use your browser to access URLs or any HTTP client in any programming language to interact with the API. Did we mention you could test drive the API with our interactive API console? Just Sign Up for an API key and start consuming this sample API right away!
